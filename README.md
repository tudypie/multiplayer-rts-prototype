# Unity multiplayer game made using Mirror and FizzySteamworks

### How to play
1. Download the Game Build zip file
2. Add the executable as a "Non-Steam Game" in your steam library
3. Host a lobby
4. Your friends can join by clicking the "join game" button in the steam friends panel

### This project was made with the help of "Unity Multiplayer: Intermediate C# Coding & Networking" Course on [Udemy](https://www.udemy.com/course/unity-multiplayer/)
